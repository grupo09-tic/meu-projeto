# Solução Escolhida e Escopo Inicial

## Solução Principal Escolhida: "Plataforma de Comunicação Interna e Feedback Centralizada via Aplicativo Mobile"

A solução escolhida integra as funcionalidades mais críticas para resolver as dores da persona Ana Clara (Colaborador) e dos outros usuários, oferecendo um alto impacto com um esforço de desenvolvimento razoável para uma fase inicial. Ela combina:

*   **Hub de Comunicação Centralizado:** Para eliminar a fragmentação e garantir que as informações importantes sejam acessíveis.
*   **Canal de Feedback Anônimo e Direto:** Para construir confiança, incentivar a honestidade e permitir que os colaboradores se expressem sem medo.
*   **Confirmação de Leitura 'Ciente' Simplificada:** Para garantir conformidade e simplificar processos administrativos para o colaborador e o RH.

---

## Escopo da Solução (Fase Inicial)

### DENTRO DO ESCOPO (IN-SCOPE):

**1. Aplicativo Mobile para Colaboradores (iOS/Android):**
    *   **Autenticação de Usuário:** Login seguro (e.g., com credenciais da empresa ou e-mail corporativo).
    *   **Feed de Comunicados:**
        *   Exibição cronológica de comunicados oficiais da empresa.
        *   Filtros básicos por categoria (e.g., RH, Gerência, Eventos).
        *   Funcionalidade de busca simples nos comunicados.
        *   Marcação de comunicados importantes como "favoritos" para acesso rápido.
    *   **Visualização de Comunicados:**
        *   Exibição do conteúdo completo do comunicado, otimizado para leitura mobile.
        *   Suporte para texto, imagens e links.
    *   **Confirmação de Leitura ("Ciente"):**
        *   Botão "Li e Concordo" ou "Ciente" proeminente nos comunicados que exigem confirmação.
        *   Registro automático da data e hora da confirmação no sistema.
    *   **Canal de Denúncias (Canal de Ética):**
        *   Formulário intuitivo para envio de denúncias **exclusivamente anônimas**.
        *   Campo para descrição textual e anexos (imagens/áudios).
        *   Remoção da funcionalidade de acompanhamento pelo colaborador (seguimento restrito ao gestor).
    *   **Módulo de Questionários:**
        *   Resposta de questionários (Clima, PDI, etc.).
        *   Visualização obrigatória de vídeo/texto de orientação antes de responder.
        *   Acesso às próprias respostas apenas em questionários configurados para tal (ex: PDI).
    *   **Módulo de RH e Documentos:**
        *   Visualização de Contracheques.
        *   Navegação hierárquica por Categorias e Subcategorias.
        *   Confirmação de "Ciente" em documentos e avisos do Mural.
    *   **Histórico de Interações Pessoais:**
        *   Seção no perfil do usuário para visualizar a lista de comunicados que leu (com status de "ciente").
        *   Listagem dos feedbacks que enviou (com o status de acompanhamento).

**2. Portal Web para Gestores de RH / Líderes de Equipe:**
    *   **Autenticação de Usuário:** Login seguro para usuários com perfil de gestor/RH.
    *   **Gerenciamento de Comunicados:**
        *   Interface para criar, editar, publicar e arquivar comunicados para os colaboradores.
        *   Definição de categorias de comunicados e públicos-alvo.
        *   Configuração para exigir "ciente" em comunicados específicos.
    *   **Acompanhamento de Confirmações ("Cientes"):**
        *   Dashboard/lista para visualizar quais colaboradores (ou grupos) confirmaram a leitura de comunicados e quais ainda não.
        *   Exportação de dados de "cientes" (e.g., CSV).
    *   **Gestão de Feedbacks/Sugestões/Denúncias:**
        *   Interface para receber, visualizar e gerenciar os feedbacks dos colaboradores.
        *   Filtros por status, categoria, tipo (anônimo/identificado).
        *   Funcionalidade básica para alterar o status do feedback (e.g., para "Em Análise", "Resolvido").
        *   Garantia da visualização do anonimato quando configurado.
    *   **Relatórios Básicos:**
        *   Métricas simples sobre a taxa de leitura de comunicados.
        *   Volume de feedbacks recebidos por tipo.

### FORA DO ESCOPO (OUT-OF-SCOPE) - Para Fases Futuras (ou para Outros Módulos/Sistemas):

*   **Gamificação:** Implementação de pontos, distintivos, rankings para incentivar o engajamento.
*   **Módulos de Avaliação de Desempenho:** Funcionalidades para ciclos de avaliação 360, gestão de metas e desenvolvimento de carreira.
*   **Integração com Sistemas Legados:** Conexões com sistemas de folha de pagamento, HRIS existentes (SAP, Workday, etc.).
*   **Chat Interno ou Rede Social Corporativa:** Funcionalidades de comunicação entre pares, criação de grupos ou perfis sociais detalhados.
*   **Módulo de Treinamento e Desenvolvimento:** Plataforma para cursos, trilhas de aprendizagem, gestão de competências.
*   **Funcionalidades Multi-tenant para Consultoria:** Gerenciamento de múltiplas empresas clientes de forma independente (foco inicial em uma única empresa).
*   **Análise de Sentimento via IA:** Processamento avançado de linguagem natural para feedbacks.
*   **Dashboards Avançados/KPIs Personalizados:** Relatórios complexos e configuráveis com análises preditivas.
*   **Notificações Customizáveis:** Configurações avançadas de notificações além das básicas de push.
*   **Edição de Documentos Online:** Ferramentas para edição colaborativa de documentos.
