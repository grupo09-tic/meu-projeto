# Front-end

Este diretório contém a interface web do sistema, desenvolvida com foco em usabilidade e experiência do usuário.

---

## Tecnologias

* Next.js
* React
* TypeScript
* Tailwind CSS

---

## Como executar

Acesse a pasta do front-end:

cd frontend

Instale as dependências:

npm install

Execute o projeto:

npm run dev

Abra no navegador:

http://localhost:3000

---

## Funcionalidades

* Tela de login
* Validação de formulário
* Modal de denúncia
* Interface responsiva
* Tela de Dashboard
---

## Estrutura

src/
├── app/
├── components/
├──hooks/
├── services/
├── utils

---

## Integração

Este front-end consome dados do backend do projeto principal.
